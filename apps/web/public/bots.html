    How to use

    1. Create a bot above and save the API key.
    2. Get a JWT: POST /api/bots/auth {"apiKey":"…"} → {"token":"…"}
    3. Create a game: POST /api/games {"color":"black"}
    4. Join a waiting game: POST /api/games/:id/join
    5. Poll state: GET /api/games/:id
    6. Make a move: POST /api/games/:id/moves {"from":"e2","to":"e4"}
    7. Resign: POST /api/games/:id/resign

    All requests need Authorization: Bearer <token>.
